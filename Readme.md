**KaanDGN Ultimate OSINT**, hedefler üzerinde derinlemesine bilgi toplamak (Intelligence Gathering) için tasarlanmış, çok fonksiyonlu ve yüksek performanslı bir keşif aracıdır. Telefon numarası analizi, sosyal medya taraması ve dijital ayak izi takibi konularında uzmanlaşmıştır.

---

## 🔥 Temel Özellikler

* **🛡️ Mega Phone Search:** Telefon numarasından operatör, ülke, hat tipi (GSM/VOIP) ve geçerlilik kontrolü.
* **🕵️ Ghost Search:** İnternetin derinliklerinde (PDF, Pastebin, Sızıntı siteleri) gizli kalmış numara izlerini bulma.
* **📱 Social Hunter:** Numaraya bağlı WhatsApp, Instagram, Twitter ve Facebook hesaplarını otomatik sorgulama.
* **🆔 Deep Nick Search:** Bir kullanıcı adını (nick) 15'ten fazla popüler platformda (TikTok, Steam, Reddit, Linktree vb.) saniyeler içinde tarar.
* **🚀 Threading Teknolojisi:** Aramalar eş zamanlı (paralel) yapıldığı için maksimum hız sağlar.
* **🔍 Active Dork Scanner:** Hedef hakkında Google Dorking kullanarak kritik sızıntıları yakalar.

## 🛠️ Kurulum ve Kullanım

Aracı kullanmak için aşağıdaki komutları sırasıyla terminalinize yapıştırın:

```bash
# Projeyi klonlayın
git clone [https://github.com/kaandgnsh/m-info-get](https://github.com/kaandgnsh/m-info-get)

# Dizine giriş yapın
cd m-info-get

# (Gerekli ise) Alt dizine giriş yapın
cd m-info-get

# Aracı çalıştırın (Gerekli kütüphaneler otomatik yüklenecektir)
python3 m-info-get.py

```
